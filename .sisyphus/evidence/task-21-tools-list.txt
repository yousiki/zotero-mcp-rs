# Task 21: Tool Router Composition + Library Auto-detect

## Verification: 15 Tools Registered

The following 15 tools are registered in `src/server.rs` via `#[tool]` methods:

1. **zotero_search** - Unified Zotero search by text, tags, citation key, or advanced conditions
2. **zotero_get_item** - Get item metadata, children, fulltext, and/or BibTeX sections
3. **zotero_update_item** - Update one item, or batch-update tags across many items
4. **zotero_delete_item** - Delete one or more Zotero items by key
5. **zotero_get_annotations** - Get all annotations for a specific item or across library
6. **zotero_get_notes** - Retrieve notes, or search notes + annotations when query is provided
7. **zotero_add_note** - Create either a Zotero note or annotation based on type
8. **zotero_list_collections** - List collections, search collections, or retrieve items in a collection
9. **zotero_manage_collections** - Create/delete collections, or add/remove items in a collection
10. **zotero_get_recent** - Get recently added items to your Zotero library
11. **zotero_list_tags** - Get all tags used in your Zotero library
12. **zotero_deduplicate** - Find or merge duplicate items in your library
13. **fetch** - ChatGPT connector fetch: return item text payload by Zotero item key
14. **zotero_add_paper** - Add a paper to Zotero by URL, DOI, arXiv ID, ISBN, or local file path
15. **zotero_rename_attachments** - Rename PDF/file attachments using Zotero's naming template


### Library Auto-detect
When `ZOTERO_LIBRARY_ID` is not set, the server calls `/keys/current` to auto-detect the user ID:
```rust
let resp = client.get("https://api.zotero.org/keys/current")
    .header("Zotero-API-Key", &config.zotero_api_key)
    .header("Zotero-API-Version", "3")
    .send().await?;
```
