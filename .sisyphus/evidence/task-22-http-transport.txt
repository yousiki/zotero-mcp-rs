```bash
MCP_TRANSPORT=http PORT=8080 zotero-mcp-rs
```

## Implementation Details

The HTTP transport uses rmcp's `StreamableHttpService` with:
- `LocalSessionManager` for in-memory session management
- Default `StreamableHttpServerConfig` (stateful mode, SSE keep-alive)
- Axum router with `/mcp` endpoint
- TCP listener bound to `0.0.0.0:{port}`

The implementation follows the MCP Streamable HTTP specification and supports:
- Session management via `Mcp-Session-Id` header
- SSE streaming for responses
- Stateful mode for client reconnection
- DNS rebinding protection via allowed hosts
